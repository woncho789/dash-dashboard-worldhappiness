## 2. HTML components overview
### Why need to learn html? 
Although HTML is not a necessary skill to use Dash, it is recommended to learn because it provides better control and hence more advance dashboard implementation. 

### How does HTML control different types of content?
HTML uses tags to define the character of contents
- <start tag> content <end tag>

### Basic HTML tags
|components| tag | dash|
|---|---|---|
|headings           |h1 - h6    | html.H1()|
|paragraph          |p          | html.P()|
|content division   |div        | html.Div()|
|line break         |br         | html.Br()|
|anchor             |a          | html.A()|

### How to use HTML is dash?
Users can use html in Dash by importing html module from dash library


## 3. Core components overview
### What is Dash Core Components (dcc) module?
The dcc is higher level components that can be interactive

### Basic dcc methods
|components| dash|
|---|---|
|dropdowns|dcc.Dropdown|
|radio items|dcc.RadioItems|
|graphs|dcc.Graph|
|text input boxes|dcc.Input|
|sliders|dcc.Slider|
|range sliders|dcc.RangeSlider|

### How to use dcc?
Users can use dcc in Dash by import dcc module from dash library

### What parameters does dcc have?
The dcc requires two core parameters, options and values
